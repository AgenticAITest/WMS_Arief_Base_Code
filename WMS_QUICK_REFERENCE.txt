KEY BUSINESS LOGIC
================================================================================
Allocation Algorithm:
  - FEFO: Orders by expiry date (earliest first)
  - FIFO: Orders by received date (oldest first)
  - Automatic quantity matching
  - Supports partial allocations

Picking Strategy:
  - Multiple picks per order item if inventory spread across locations
  - Batch/Lot/Serial number tracking
  - Expiry date preservation through workflow

Workflow State Machine:
  - Sales Order: 7 states (created→allocated→picked→packed→shipped→delivered→complete)
  - Purchase Order: 5 states (pending→approved→rejected→incomplete→received→completed)
  - Custom workflows per tenant

Document Generation:
  - Configurable numbering formats
  - HTML document generation
  - PDF storage capability
  - Historical tracking
